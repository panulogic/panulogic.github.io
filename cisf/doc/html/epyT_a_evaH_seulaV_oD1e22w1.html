Do Values Have a Type?

Trick Question

Values don't have A type!

They have many  types

 

  • 123 is an instance of Number

     

  • 123 is an instance of PositiveNumber

     

  • 123 is an instance of BiggerThan100

     

  • 123 is an instance of LessThan125

     

  • 123 is an instance of Type (Odd, PositiveNumber, ...)

 

In JavaScript every value
besides null and undefined
have a constructor.

A constructor can be used as a type
whose instances are all the values
whose constructor is that constructor.

The constructor is what we often
think of as the 'type' of a value.
But it is just one of them.
