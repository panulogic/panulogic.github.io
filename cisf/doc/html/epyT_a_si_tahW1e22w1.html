What is a Type?

A Type is anything that can be used to
divide ALL values into TWO SETS:

 

  • Instances of the type
  • Everything else

 

'Type' is much like Set in Set-theory.
Either you're in the set or you're not.

Every value is either a String
or not a String.

=> String is a Type.

 

What is a "Value" ?

Anything you can store into a variable

 

Type !== TYPE DECLARATION

Type !== TYPE-CHECKING

'Type' is an abstract concept.

 

We can represent types
as Functions in JavaScript
