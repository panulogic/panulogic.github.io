fails()

Takes as argument a Function.

 

  • If calling the function
  • causes an Error
  • fails() returns that error.

 

  • If calling the function
  • does NOT cause an Error
  • fails() causes an error.

 

fails()
1 fails (_ => "s".noSuch());
2 fails (_ => fails ( _=> 123 ));
3
4 x (fails (_ => "s".noSuch()), Error);
 

Useful for documenting an API to show what arguments are NOT allowed. For example, documenting Cisf.js.
