Predicate Types

To create a Predicate type
  • Call Type() with a function
  • which has a name
  • which does not start with [A-Z]:

 

Instances of the created "Predicate Type"
are values for which the function returns true.

Predicate-Type 'Odd':
1 let Odd = Type (odd_p);
2
3 function odd_p (n)
4 { if (! arguments.length)
5 { return 7;
6 }
7 if ( Math.abs (n % 2) === 1 )
8 { return true;
9 }
10 }
11
12 x (1 , Odd )
13 x (-1 , Odd )
14 not (is(2 , Odd))

  • Type(_=> 123) creates a Function-Type
